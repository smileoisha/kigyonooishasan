# スキル：社内・自己MTG準備 (/mtg-prep)
## 使い方
`/mtg-prep` で起動（引数なし）
## 手順
1. Google Calendarで本日・翌日の予定を取得
2. 各予定の内容を確認し、準備が必要なものを特定
3. 顧客名が含まれる場合は /deal-prep と同様の情報収集を実施
4. 以下のフォーマットで出力
## 出力フォーマット
### 本日・翌日のMTG一覧
| 時間 | 相手 | 種別 | 準備状況 |
|------|------|------|---------|
|      |      |      |         |
### 各MTGの準備メモ
（MTGごとに確認ポイントを箇条書き）
